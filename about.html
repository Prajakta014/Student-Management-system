📘 About This Project

The Student Management System is a responsive, feature-rich web application designed to efficiently manage student records. Built with HTML, CSS, and JavaScript, it includes robust backend support using MySQL or localStorage for data persistence.

  • 📋 Full CRUD Operations: Easily Add, View, Edit, and Delete student data
  • 🌙 Light/Dark Theme Toggle: Enhance user experience with a simple switch
  • 💾 Data stored securely using localStorage or integrated MySQL Database
  • 📷 Upload & manage student profile photos for identification
  • 📊 Export records to CSV format for reports and backups
  • 🔐 Secure Login & Registration System with field validations
  • 🛠️ Modular & Scalable Design: Easily expandable for future enhancements
